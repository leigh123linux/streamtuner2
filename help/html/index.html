Streamtuner2

Streamtuner2 shows internet radio stations for easy browsing, playing, recording.

Introduction — Basic usage instructions. Selecting a channel, category, and playing a radio.

Browsing channels and radio stations

Channels — Switching through the channel tabs, different radio directory services.
Playing — Double click a station to start it in your audio player.
Streams / Stations — Radio streams lists, station information columns, entry actions.

Channel tabs

Bookmarks — Collect favourites via bookmark function, entry editing and deleting.
Internet-Radio.com — Large radio directory from the UK.
Jamendo — Creative Commons music and artist hub.
Live365 — Live365 radio network
MyOggRadio — Open source stream directory.
Shoutcast — One of the bigger radio station lists.
SurfMusic — International radio listings.
The MOD Archive — MOD files.
TuneIn — Radio and Podcast directory.
Xiph.org — ICEcast radio directory.
Youtube — Youtube music/video listings.
iCast.io — iCast
iTunes Radios via RS — iTunes (RoliSoft Playlists)

Functions

Radio homepages — Start a web browser for a station.
Recording — Save radio songs as MP3 files via streamripper.
Saving — Export a station entry as .m3u/.pls file.
Searching — Quicksearch field and Ctrl+F compound search window.

Configuration

Settings dialog — F12 brings up the options window with the Player, Display, System and Plugin sections.
