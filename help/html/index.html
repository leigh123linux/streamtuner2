Streamtuner2

Streamtuner2 shows internet radio stations for easy browsing, playing, recording.

Introduction — Basic usage instructions. Selecting a channel, category, and playing a radio.

Channels — Switching through the channel tabs, different radio directory services.
Playing — Double click a station to start it in your audio player.

Functions

Edit — Edit or inspect a station entry.
Radio homepages — Start a web browser for a station.
Recording — Save radio songs as MP3 files via streamripper.
Saving — Export a station entry as .m3u/.pls file.
Searching — Quicksearch field and Ctrl+F compound search window.
